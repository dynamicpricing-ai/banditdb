What the numbers mean

theta_norm
L2 norm of the weight vector θ per arm. Higher = stronger learned signal. A freshly created campaign shows near-zero values; after hundreds of predict→reward cycles the arms diverge as the model learns which context profiles each arm serves best.
Predictions / Rewards
How many times each arm was selected, and how many of those received a reward. A gap means interactions are still in-flight — reward not yet sent. The reward bar shows the fill rate.
Algorithm
LinUCB adds a deterministic UCB bonus: score = θ·x + α·√(x·A⁻¹·x). It always picks the arm with the highest upper confidence bound.

Thompson Sampling samples θ̃ from the posterior N(θ, α²·A⁻¹) — concurrent users naturally diversify arm coverage.
Context vector
Fixed-length array of floats in [0, 1] describing the current situation. The model learns which arms work for which context profiles — personalisation emerges from the predict→reward signal.
